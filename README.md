## Installation

Use the package manager [node.js](https://nodejs.org/en) to install nodejs
and [vite.js](https://vite.dev/) to install vite.js

```bash
git clone https://github.com/Pimmawat/codingProject.git
cd server
npm install
npm run dev 
cd client
npm install
npm run dev
```
